# C++ 算法学习与练习

这是一个 C++ 算法学习和练习的仓库，包含各种算法题目解答和常用算法模板。

## 仓库描述

本仓库主要用于算法学习和竞赛编程练习，包含了多个在线题平台的题目解答，以及各种常用算法和数据结构的模板实现。

## 目录结构

- **LeetCode/** - LeetCode 算法题解答
  - 包含数组、链表、树、动态规划等各类算法问题的 C++ 实现
  
- **Luogu/** - 洛谷平台题目解答
  - NOIP 系列题目
  - 各类算法竞赛题目
  
- **HZOJ/** - HZOJ 在线评测系统题目解答
  
- **Templates/** - 常用算法和数据结构模板
  - 动态规划模板：01背包、完全背包、多重背包、分组背包
  - 排序算法：快速排序、归并排序
  - 数据结构：二叉树、AVL树、红黑树、B树
  - 图算法：Dijkstra、Prim、LCA、树链剖分
  - 字符串算法：KMP、Rabin-Karp
  - 其他：二分查找、并查集、约瑟夫环等
  
- **Others/** - 其他练习题目和测试代码
  
- **class/** - 课程相关代码
  - 数据结构课程作业和实验

## 主要技术栈

- **编程语言**: C++
- **编译器**: 支持主流 C++ 编译器
- **开发环境**: Visual Studio Code

## 特点

- 涵盖多种经典算法和数据结构
- 包含多个知名在线题平台的题目解答
- 提供高质量的算法模板，方便快速开发
- 代码结构清晰，便于学习和参考

## 使用说明

1. 克隆仓库到本地
2. 根据需要浏览不同目录下的代码
3. 参考模板实现自己的算法
